# 🚀 OtoInstall — AI Deploy

**Deploy your AI-generated projects to live servers in seconds.**

Right-click any folder → Deploy. Works with any hosting via FTP / SFTP / SSH.

Perfect for **Vibe Coding** — build with AI, deploy instantly.

## 🎯 Quick Start

### 1. Install the Extension

Search "OtoInstall" in VS Code Extensions or install from [Marketplace](https://marketplace.visualstudio.com/items?itemName=otoinstall.otoinstall).

### 2. Get Your API Key

Sign up at [otoinstall.com](https://otoinstall.com) → Dashboard → API Keys → Create Key

### 3. Configure

- Open Command Palette (`Ctrl+Shift+P`)
- Type "OtoInstall: Set API Key"
- Paste your API key

### 4. Deploy!

- **Right-click** any folder in Explorer → "🚀 Deploy Project"
- **Or** press `Ctrl+Shift+D`
- **Or** click the Deploy button in the status bar

---

## 📸 How It Works

1. **Select Server** — Choose from your configured FTP/SFTP/SSH servers
2. **ZIP & Upload** — Your project is automatically zipped and uploaded
3. **Security Scan** — OtoInstall scans for vulnerabilities
4. **Auto-Fix** — Issues are automatically resolved
5. **Deploy** — Files are deployed to your live server
6. **Verify** — OtoInstall verifies the deployment

All in under 30 seconds! ⚡

## 🛠️ Available Commands

Open Command Palette (`Ctrl+Shift+P`) and type "OtoInstall":

| Command | Description |
|---|---|
| `OtoInstall: Deploy Project` | Deploy current workspace |
| `OtoInstall: Deploy ZIP File` | Deploy an existing ZIP file |
| `OtoInstall: Check Deploy Status` | Check deployment progress |
| `OtoInstall: View Deploy Logs` | View detailed deployment logs |
| `OtoInstall: List Servers` | Show configured servers |
| `OtoInstall: List Projects` | Show recent deployments |
| `OtoInstall: Set API Key` | Configure your API key |

## ⚙️ Settings

| Setting | Default | Description |
|---|---|---|
| `otoinstall.apiKey` | — | Your OtoInstall API key |
| `otoinstall.baseUrl` | `https://otoinstall.com` | API base URL |
| `otoinstall.autoDetectProjectType` | `true` | Auto-detect project type |
| `otoinstall.showStatusBar` | `true` | Show deploy button in status bar |
